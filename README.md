# XsnapImaging

XsnapImaging is a lightweight web front-end for showcasing medical and scientific imaging projects. The site provides informative pages, image upload support, and a simple, responsive layout so researchers and clinicians can present imaging results, products, and documentation.

Key features
- Clean, responsive HTML/CSS site for image-focused content
- Image upload directory for sharing or demoing sample scans (`uploads/`)
- Pages included: home (`index.html`), product (`product.html`), and about (`aboutus.html`)

Getting started
1. Open `index.html` in a browser to view the site locally.
2. Place images into the `uploads/` folder to reference them from pages.
